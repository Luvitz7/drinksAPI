Borracho Refinado

¡Bienvenido a esta pequeña aplicacion!

Descripcion rapida: Aquí podras hacer consultas de bebidas de todo tipo, alcholicas o sin alcohol. Para cualquier consulta solo usar los espacios de busqueda a continuación. Puedes buscar por nombre y seleccionar alguna bebida de los resultados obtenidos. Si llegas a tener duda de algun ingrediente puedes usar la siguiente sección recibiras informacion sobre el ingrediente. Todas las consultas hacerlas en ingles (de momento)

Esta aplicacion hace consultas a la API de The CoctailDB puedes visitarla haciendo click en el nombre. En la parte de abajo podras encontrar el enlace a Github para ver el código.
